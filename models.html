TBH.AI Vortex Reasoning

Experimental reasoning model with SFT approach

Featured

TBH.AI Vortex Reasoning is an experimental model that advances structured reasoning capabilities without using Generalized Reinforced Policy Optimization (GRPO). Instead, it relies on high-end Supervised Fine-Tuning (SFT) techniques to achieve similar self-reflective thought processes, proving that structured reasoning can be induced through high-quality SFT alone.

Model Type Language Model
Parameters 3.09B
Base Model Qwen2.5-3B
License Apache-2.0

Key Features

Self-Questioning & Iterative Thinking

Actively asks itself intermediate questions before answering, mimicking deep reflection-based thought processes

High-Quality SFT Dataset

Trained on mathematical proofs, logical puzzles, and complex problem-solving tasks

Implicit Reasoning Markers

Uses special reasoning markers to structure responses for consistent and methodical answers

Security Implementation

Security features implemented on a small scale using internal security datasets

TBH.AI Base Reasoning

Structured reasoning model with step-by-step problem solving

TBH.AI Base Reasoning is a specialized model that uses thinking tokens similar to Deepseek R1 to enhance reasoning capabilities. This model is designed to break down complex problems into manageable steps, providing clear and logical thought processes in its responses.

Model Type Language Model
Parameters 3.09B
Base Model Qwen2.5-3B
License Apache-2.0

Key Features

Thinking Token Approach

Uses specialized tokens to structure reasoning similar to Deepseek R1

Step-by-Step Problem Solving

Breaks down complex problems into logical steps for clearer reasoning
